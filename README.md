# EyeTrackingVisualization

## Prerequisites
- [Node.jS](https://nodejs.org/en/download/)

Ensure **Node.js** and **npm** are installed.
```bash
node -v
npm -v
```

## Setting up
Installing packages:
```bash
npm i
```

Start script:
```bash
npm start
```

Access locally hosted website through:
http://localhost:3000/
